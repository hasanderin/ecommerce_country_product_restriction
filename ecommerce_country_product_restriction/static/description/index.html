Overview

With this module, users can restrict their product's availability for specific countries. For e.g., if product A is available in United States but is unavailable in Germany, in such cases, this module will disable the payment option for the user wanting to buy the product A in Germany

Features

Configure product availability for particular location.

Customize country list where products will be made available

Customize the message to display upon product unavailability.

User can't checkout if particular product isn't available at selected country.

Screenshots

Product Form View

  • We added a page "Countries" here .
  • User can select the countries here where the product is available.
  • If the user has not selected any country then that product is available for all countries.

Website Form View

  • We added a page "Product Country Configuration" here .
  • User can select the country/countries which will be appearing in the country selection of checkout address as options.
  • User can also add customised message in checkout message and cart message which will be appearing as a warning when product is not available.

Website Shop Cart View

  • If the product is not available in user's country then it will display warning in the product cart lines.
  • In case the user is a Guest User, system will not show him any warning on the cart page. It will show him the warning once he has entered his shipping address.

Website Checkout View

  • If any Product added to the Cart is not available for the user's shipping country then system will display a warning message on the checkout page.

  • The Payment button will be disabled in case any product is not available for the shipping address country.
